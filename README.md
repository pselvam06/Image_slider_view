# React Image Slider (Nature + Animals)

This project contains a stylish React image slider page with:

- 20 local sample images (10 nature + 10 animal)
- Next / Previous controls
- Loop mode navigation
- Modal preview view
- Keyboard support (`ArrowLeft`, `ArrowRight`, `Escape`)

## Run

```bash
npm install
npm run dev
```

Open the local URL shown in terminal.
